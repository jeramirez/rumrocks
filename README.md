# rumrocks
========
rumrocks master repository

Requirements
------------

```shell
sudo dnf -y install epel-release
sudo dnf -y install apptainer
```

Download
--------

```shell
git clone https://github.com/jeramirez/rumrocks.git
cd rumrocks
sudo singularity build rumrocksapptainer.img containerbuilder
./rumrocksshell.sh
.
.
in progress
```
